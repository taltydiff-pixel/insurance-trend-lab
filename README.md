# 险业热点灵感台

一个可独立部署的保险热点采集与 AI 内容策划网站。服务器端采集最近公开信息，DeepSeek负责筛选、事实边界控制与内容策划。

## 本地运行

1. 安装 Node.js 20+
2. 运行 `npm install`
3. 复制 `.env.example` 为 `.env.local`，填入 `DEEPSEEK_API_KEY`
4. 运行 `npm run dev`
5. 打开 `http://localhost:3000`

## 部署

可直接导入 Vercel。部署时添加环境变量 `DEEPSEEK_API_KEY`。如需更稳定、覆盖更广的检索，可再配置可选的 `TAVILY_API_KEY`。

## 数据边界

- 默认以公开新闻 RSS 为检索入口；配置 Tavily 后使用增强检索。
- 抖音、小红书仅能采集公开可索引页面，不能替代平台官方热榜或登录后的站内数据。
- 页面保留原始来源、发布时间、采集时间和置信标识。
- 政策、收益和保险条款必须回看官方原文或合同。
